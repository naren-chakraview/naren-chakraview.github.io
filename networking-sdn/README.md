# Chakraview Networking-SDN

A three-layer portfolio project: DPDK/eBPF packet processing, Go SDN controller,
Python fabric protocols (BGP/VXLAN/EVPN), Docker Compose lab.

## Quick Start

```bash
cd lab
docker-compose up
./scripts/init.sh
```
